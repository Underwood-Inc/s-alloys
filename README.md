# Alloys — public site

Player-facing guide and interactive test checklist for the **Alloys** Minecraft 26.2 datapack.

Live: [underwood-inc.github.io/s-alloys](https://underwood-inc.github.io/s-alloys/)

## What this repo contains

- **Guide** — install steps, alloy reference, crafting expectations, checklist how-to
- **Test checklist** — installable PWA with CSV/JSON import and export for team playtests

This repository intentionally contains **no private source code**. Test case text is sanitized before publish.

## Local development

```powershell
cd s-alloys
npm install
npm test
npm run dev
```

Open the URL Vite prints (typically `http://localhost:5173/s-alloys/`).

## Architecture (CLAD)

| Tier | Location | Role |
|------|----------|------|
| Atoms | `src/atoms/` | Pure helpers (CSV escape, element registry) |
| Molecules | `src/molecules/` | Session store, filters, CSV exchange |
| Organisms | `src/organisms/` | Web components (site header) |
| Recipes | `src/recipes/` | Checklist runner wiring |
| Views | `src/views/` | Route-level custom elements |
| App | `src/app/` | Shell + router |
| Plugs | `src/plugs/browser/` | Browser storage adapter |

### Recipe explorer (CLAD)

| Tier | Path | Role |
|------|------|------|
| Molecule | `src/molecules/crafting-model/` | Agnostic shaped/shapeless grid model |
| Molecule | `src/molecules/recipe-catalog/` | Alloys canon: layouts, ingredients, `buildRecipeForTab()` |
| Molecule | `src/molecules/tooltip-model/` | Legendary-style game tooltips |
| Organism | `src/organisms/crafting-grid/` | `<crafting-grid>` — 3×3 + result slot |
| Organism | `src/organisms/recipe-explorer/` | `<recipe-explorer>` — alloy rail + image category nav |

Trace IDs live in `requirements-registry.yaml`. Run `npm test` before publishing.

## Assets (committed — not generated by build)

All site media lives under **`public/`** and is edited in place. `npm run build` only emits `dist/`; it does **not** regenerate `public/`.

| Path | Role |
|------|------|
| `public/icons/frames/{alloy}.png` | Hero / nav alloy crossfade (`<alloys-site-icon>`) |
| `public/icons/icon-{64,192,512}.png` | Favicon / PWA |
| `public/guide/chapters/` | Guide chapter art |
| `public/guide/nav/` | Recipe category nav tiles |
| `public/guide/ui/` | Crafting grid chrome |
| `public/guide/ingots`, `gear`, `ingredients`, `fragments` | Recipe explorer sprites |

Marketing frame sources (private repo): `docs/media/icon-frames/` — copy into `public/icons/frames/` when updating hero art.

Optional ambient WebM: `public/guide/media/slot-shimmer.webm` (CSS shimmer is the default).

## Publishing checklist data (private maintainers only)

From the private **alloys** repository:

```powershell
node tools/_export-public-s-alloys.mjs
```

That updates **`public/data/cases.json` only**, then runs `npm run build`. Icons and guide art are never touched by export.

## Deploy

GitHub Pages serves the built `dist/` output. Use the included GitHub Actions workflow or run `npm run build` and publish `dist/` per your Pages settings.
